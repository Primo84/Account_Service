#Account_Backend  oraz Account_Fromntend

Podstawowa wresja systemu tworzenia oraz logowania kont użytkowników.
Daje mozliwość tworzenia własngo serwisu gdzie wykorzystywane są konta zarajestrowanych użytkowników.
Oparty jest na bazie Postgresql skonfigurowanej lokalnie. 
Parametetry połączenia znajdują się w pliku Account_Backend/Postgres_DB.js
